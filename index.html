더빙 작업 만들기

아직 작업이 시작되지 않았어.
현재는 프론트 화면만 먼저 완성한 상태야. 다음 단계에서 이 버튼에 실제 API를 연결하면 진짜 더빙 요청 흐름으로 바꿀 수 있어.

프로젝트 개요

이 페이지는 방문자에게 서비스 목적과 현재 상태를 바로 보여주는 랜딩 + 작업 시작 화면 역할을 해.

업로드 준비 파일 선택, 포맷 확인, 언어 선택까지 한 화면에서 처리
확장 쉬움 버튼 이벤트에 API 연결만 추가하면 실제 서비스 구조로 확장 가능
Vercel 친화적 HTML 하나만 수정해도 바로 자동 재배포되므로 관리가 편함
포트폴리오용 가능 초기 버전부터 서비스처럼 보이는 화면으로 보여줄 수 있음
